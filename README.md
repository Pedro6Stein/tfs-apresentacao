# 🐸 Three Frog System (TFS) - Governança de TI

Este projeto é uma plataforma de apresentação interativa desenvolvida para a disciplina de **Gestão de TI na FATEC**. O objetivo é apresentar o caso de negócio da TFS, focando na implementação do framework **COBIT 2019**, conformidade com a **LGPD** e estratégias de escalabilidade de software.

---

## 🚀 Diferenciais Técnico

Diferente de apresentações comuns, este projeto foi construído como um **Dashboard Executivo** em React, focado em alta legibilidade para TVs de grande formato (55"+).

* **Aspect Ratio Scaler:** Sistema customizado que trava a proporção em **16:9**, garantindo que o design nunca quebre, independentemente da resolução do monitor ou TV.
* **Animações Cinemáticas:** Uso de `framer-motion` com curvas de interpolação baseadas no design system da Apple para transições suaves e profissionais.
* **Tailwind CSS v4:** Implementação utilizando a última versão do motor de estilos, garantindo performance e um visual *Dark Mode* profundo com detalhes em *Neon Blue*.
* **Offline Ready:** Todas as dependências (ícones, fontes e lógica) são empacotadas no build final, permitindo a execução sem internet no dia da banca.

---

## 📂 Conteúdo da Apresentação

O roteiro segue a estrutura técnica de Governança de I&T:
1.  **Caso de Negócio:** Foco em Java e NextJS para otimização de processos.
2.  **Stakeholders:** Necessidades de transparência, previsibilidade e responsabilidade.
3.  **Conformidade:** Estratégias para LGPD, ISO/IEC 27001 e normas do BACEN/CVM.
4.  **COBIT 2019:** Ciclo de vida de 7 fases e aplicação dos domínios (EDM, APO, BAI, DSS, MEA).
5.  **Matriz RACI:** Definição clara de papéis e responsabilidades.

## 📦 Como Rodar o Projeto

1.  **Instale as dependências:**
    ```bash
    npm install
    ```
2.  **Inicie o servidor de desenvolvimento:**
    ```bash
    npm run dev
    ```
3.  **Para o Build de produção (Offline):**
    ```bash
    npm run build
    ```
